
展示型组件
关心数据的展示方式;
不依赖app中的其它文件;
不关心数据是如何加载和变化的;
仅通过props接受数据和回调函数;
除非需要用到state生命周期函数或性能优化，通常写成函数式组件.

容器型组件
关心数据运作方式;
为展示型组件提供数据和操作数据的方法;
为展示型组件提供回调函数;
通常是有状态的，并且作为数据源存在.

components文件夹存放所有的展示型组件
containers文件夹存放所有的容器型组件
这两个文件夹下的__test__文件夹存放所有的组件测试文件
文件使用pascal case格式
src根目录下放置统一使用的一些文件。
